🔒 番號達人 隱私政策

最後更新：2026 年 7 月 11 日  ·  版本 1.2.0

1. 概述

番號達人（以下簡稱「本擴充功能」）是一款 Chrome 擴充功能，用於自動識別特定成人影片資訊網站中的番號，並顯示對應的片名、女優、發行商等詳細資訊。

我們承諾：本擴充功能不收集、不傳送、不出售任何使用者個人資料。

2. 資料存取範圍

本擴充功能會在所有網站執行內容腳本，以自動偵測頁面文字中出現的番號（如 SSIS-001、IPX-123），並將其標記顯示。這是核心功能所必需，因為番號可能出現在任何網站（論壇、社群媒體、搜尋結果等）。

內容腳本僅執行以下操作：

  • 掃描頁面純文字節點，比對符合番號格式的字串
  • 為確認番號是否有效，會於掃描時自動向番號資訊網站送出「存在性查詢」（僅傳送番號字串），有效者才加上視覺標記（黃色高亮）
  • 使用者滑鼠懸停於標記番號時，才進一步查詢並顯示完整詳細資料（片名、女優、封面等）

內容腳本不讀取、不傳送任何網頁內容、表單資料或個人資訊至任何伺服器。

番號資訊查詢僅向以下網站發送請求（透過 Service Worker）：

  • host javdb.com / www.javdb.com
  • host javbus.com / www.javbus.com
  • host javlibrary.com / www.javlibrary.com
  • host mgstage.com / www.mgstage.com

3. 本地快取

為提升查詢速度，本擴充功能會將以下資料儲存在 Chrome 本地儲存空間（chrome.storage.local）：

  • 番號查詢結果（片名、女優、發行商、封面圖片 Base64）
  • 使用者偏好設定（啟用/停用番號偵測、Hover 查詢）

上述資料僅存放在您的裝置本地，不會上傳至任何伺服器。快取有效期為 3 天，超過上限會自動刪除最舊的紀錄。您也可以隨時透過擴充功能彈出視窗手動清除。

4. 外部網路請求

本擴充功能的 Service Worker 會於偵測到番號時（掃描階段的存在性查詢），以及使用者懸停時（詳細資料查詢），向以下網站發送 HTTP 請求以取得番號資訊：

  • javdb.com
  • javbus.com
  • javlibrary.com
  • mgstage.com

這些請求僅包含番號字串，不包含任何個人識別資訊，且不會攜帶您在上述網站的登入 Cookie（以匿名方式查詢，避免使用者身分被關聯）。

5. 申請的權限

  • storage 用於本地儲存快取與偏好設定
  • host_permissions 用於向上述番號資訊網站發送查詢請求

本擴充功能遵循最小權限原則，不申請任何非必要權限。

6. 第三方服務

本擴充功能在查詢番號時會存取第三方網站（JAVDB、JAVBUS 等），使用者的請求將受到這些網站各自的隱私政策約束。本擴充功能對第三方網站的行為不負任何責任。

7. 未成年人保護

本擴充功能涉及成人內容，嚴禁未滿 18 歲（或依據當地法律規定之法定年齡）之人士使用。

8. 免責聲明

本工具僅供學習與技術研究使用。使用者須自行遵守相關網站之服務條款及當地適用之法律法規。開發者對使用本擴充功能所產生的任何後果不承擔責任。

9. 政策變更

若本隱私政策有任何重大變更，將更新本頁面的「最後更新」日期並於新版本說明中告知。

10. 聯絡方式

如有任何隱私相關疑問，請透過 Chrome Web Store 的「聯絡開發者」功能聯繫我們。
