# 番號達人 Chrome 擴充功能

## 📋 專案概述

番號達人是一個 Chrome 擴充功能，能夠自動識別網頁中的番號並提供詳細資訊查詢功能。支援多個資料來源，查詢優先級如下：

| 順序 | 資料來源 | 特色 |
|------|----------|------|
| 1 | JAVLibrary | 中文資料最準確 |
| 2 | JAVDB | 中文補充 |
| 3 | JAVBus | 備援 |
| 4 | MGStage | 日本官方 |
| 5 | 3xplanet | 日文片名、女優、片商 |
| 6 | JAV Database | 完整英文資料 |
| 7 | JavMost | 英文最後備援 |

## ✨ 主要功能

- 🔍 **自動番號識別**：掃描網頁內容，自動標記番號
- 📊 **詳細資訊顯示**：懸停番號顯示片名、女優、發行商等資訊
- 🖼️ **圖片快取**：智能快取封面圖片，提升載入速度
- 📋 **一鍵複製**：點擊番號即可複製到剪貼簿
- ⚡ **高效能**：優化的 DOM 掃描和快取策略
- 🔒 **安全防護**：XSS 防護和輸入驗證

## 🛠 技術架構

### 核心檔案
- `manifest.json` - 擴充功能配置檔案
- `background.js` - Service Worker，處理資料獲取和快取
- `content.js` - Content Script，處理 DOM 操作和用戶交互
- `popup.js` - 彈出腳本，處理設定管理
- `styles.css` - 樣式檔案

### 安全特性
- ✅ XSS 防護：使用 `textContent` 替代 `innerHTML`
- ✅ 輸入驗證：嚴格的番號格式驗證
- ✅ 權限最小化：僅申請必要的 host_permissions，所有來源均為 HTTPS
- ✅ 錯誤處理：統一的錯誤處理機制
- ✅ 記憶體管理：防止記憶體洩漏的快取策略
- ✅ 隱私政策：詳見 [privacy.html](privacy.html)

### 效能優化
- ✅ 圖片壓縮：自動檢查和限制圖片大小
- ✅ DOM 掃描節流：避免過度掃描
- ✅ 智能快取：LRU 快取演算法
- ✅ 批次處理：減少 DOM 操作頻率

## 📦 安裝說明

### 開發模式安裝
1. 克隆或下載此專案
2. 開啟 Chrome 擴充功能管理頁面 (`chrome://extensions/`)
3. 開啟「開發者模式」
4. 點擊「載入未封裝項目」
5. 選擇專案根目錄

### 打包發布
1. 在擴充功能管理頁面點擊「打包擴充功能」
2. 選擇專案根目錄
3. 生成 `.crx` 檔案

## ⚙️ 設定選項

透過擴充功能圖示右鍵選單或彈出視窗可以調整：
- 🔘 啟用/停用番號偵測
- 🔘 啟用/停用懸停查詢
- 🗑️ 清除快取資料

## 🔧 開發指南

### 環境需求
- Chrome 88+ (支援 Manifest V3)
- 基本的 JavaScript 和 HTML/CSS 知識

### 調試模式
在 `content.js` 和 `background.js` 中將 `DEBUG_MODE` 設為 `true` 可開啟詳細日誌。

### 程式碼規範
- 使用 ES6+ 語法
- 統一的錯誤處理機制
- 防止 XSS 攻擊的安全編碼實踐
- 記憶體洩漏防護

## 🐛 問題修復記錄

### v1.2.0 安全性強化（2026-07-11）
- ✅ 查詢改為匿名請求（`credentials: 'omit'`）：不再攜帶使用者在番號網站的登入 Cookie，避免任意頁面藉埋入番號字串進行去匿名化
- ✅ 自動查詢速率限制：單頁單批最多查詢 50 個番號，防止惡意頁面觸發海量請求
- ✅ 隱私政策更新：如實揭露「存在性查詢於掃描時自動觸發（非僅懸停）」且請求不含登入 Cookie
- ℹ️ content_scripts 維持 `<all_urls>`：核心功能需在任意網站偵測番號，故無法限縮；風險已由上述匿名查詢消除，並於隱私政策充分揭露

### v1.2.0 新增資料來源（2026-06-23）
- ✅ 新增 javbooks.com host_permissions（頁面番號標記支援）
- ✅ 新增 fetchJavmost()：英文片名、女優、片商（javmost.ws）
- ✅ 新增 fetchJavdatabase()：完整英文資料（javdatabase.com）
- ✅ 新增 fetchThreexplanet()：日文片名、女優、片商（3xplanet.com）
- ✅ 查詢優先級優化：7 層備援確保最高覆蓋率

### v1.0.1 上架前合規修正（2026-06-22）
- ✅ 新增 privacy.html 隱私政策頁面（Chrome Web Store 上架必要）
- ✅ 縮小 content_scripts matches：從 `<all_urls>` 改為指定目標番號網站（⚠️ 已於後續版本因核心功能需求還原為 `<all_urls>`，詳見上方 v1.2.0 安全性強化）
- ✅ 修正 content.js 頂部架構說明註解錯誤（5種特殊字符，OWASP 標準）

### v1.0.0 安全與效能更新

## 🔒 隱私與安全

- 僅存儲必要的快取資料在本地
- 不收集或傳送任何個人資訊
- 使用最小權限原則：Content Script 僅注入目標番號網站
- 定期清理過期快取
- 詳細隱私政策請見 [privacy.html](privacy.html)
